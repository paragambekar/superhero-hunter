
# Super Hero Hunter

Search superhero from the API and display the results of the response from the API.



## Features

- Home Page

  Search superhero and the results can be viewed on the home Page

  Can add a heroes to our favorite list 

  Can view additional information about a hero by clicking on More info button

- SuperHero Page

  View detailed information about the superhero eg: powerstats,apperance,publisher,firstapperance,etc.


- Favorite SuperHero Page

  Display list of all favorite superheroes

  The list is persistent i.e. it will contain the info even when the browser if restarted.

  Can remove a hero from our favorite superhero list
